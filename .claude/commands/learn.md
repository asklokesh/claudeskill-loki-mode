# Update Team Playbook

Update CLAUDE.md with new learnings (Boris Cherny: "Our team updates CLAUDE.md multiple times a week").

## Tasks
1. Identify what was learned in recent work
2. Document any fixes for common mistakes
3. Add useful patterns discovered
4. Update architecture decisions if relevant

## Format
Add to CLAUDE.md:
```markdown
#### [timestamp]
**Learning**: [what was learned]
**Context**: [when this applies]
**Fix/Pattern**: [the solution or pattern]
```
